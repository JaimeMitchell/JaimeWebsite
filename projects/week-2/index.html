Git and Github had been something I've had some experience with over the past 6 months. As part of my Corona lock down I bought a Raspberry Pi and Elegoo set (Arduino kit knockoff) and joined a free EdX class called CS50 to learn more about C, Command line and Python in order to really understand embedded code to create electronic projects, and not just copy and paste it.

CS50 required joining Github and I succeed in setting up an account and using it to send my finished assignments in, but didn't understand what the rest of github was actually for.

Not until I had the old desktop I was using die on me during my website build last week. I had backed up my work on google drive but that was an older version, so the ability to have a program (git) that saves every move I make on a cloudbased platform (Github), became paramount in keeping accurate and automatically updated documentation of my fabrication work in this program,

Github also hosts my website for free.
